## Dotfiles
Version-controlled collection of Linux configurations. Note that this contains
no installations or data, so you'll have to do some work to make this all
functional.

### Installations
Bootstrapping:
* nvm (node/npm/npx)
* pipx
* rustup (cargo)

With npx:
* copilot

With pipx:
* jrnl
* userpath

With cargo:
* bat
* btm
* btop
* choose
* dua
* fd-find
* just
* rg
* skim
* starship

From source:
* foot
* mako
* swayfx
* waybar
* wofi

Fonts (powerline/nerdfont):
* Hack
* Noto Color Emoji
